This plugin is for WordPress developers looking to tap into the power of Windows Azure from within WordPress plugins.

To install this plugin simply copy the folder into your WordPress plugins folder, 
visit the Plugins page in wp-admin and activate the plugin. The Windows Azure SDK for PHP 
library will now be available for us in plugins.

You are now able to immediately start utilizing the classes in the Windows Azure SDK for PHP.

For information on using the Windows Azure SDK for PHP see the following links:

* http://azurephp.interoperabilitybridges.com/tutorials
* http://phpazure.codeplex.com/
